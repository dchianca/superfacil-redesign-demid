# SuperFácil Redesign

Projeto desenvolvido para a disciplina **Internet IV**, do curso de **Comunicação em Mídias Digitais da Universidade Federal da Paraíba (UFPB)**.

O objetivo deste trabalho foi realizar o **redesign completo do site institucional do SuperFácil Atacado**, modernizando sua interface, reorganizando a arquitetura da informação e aprimorando a experiência do usuário por meio da aplicação de conceitos de **UI/UX Design**, **Design Responsivo** e **Desenvolvimento Front-end**.

Todo o projeto foi desenvolvido utilizando **HTML5**, **CSS3**, **JavaScript** e **Bootstrap 5**, preservando a identidade visual da marca enquanto propõe uma navegação mais intuitiva, moderna e acessível.

## 1. Página Inicial (Home)

A página inicial do site original apresentava excesso de informações, baixa hierarquia visual e um layout visualmente desatualizado, dificultando que o usuário encontrasse rapidamente os principais conteúdos.

No redesign, a Home foi completamente reorganizada, priorizando uma navegação mais intuitiva, maior destaque para ofertas e serviços, além de uma distribuição mais equilibrada dos elementos da interface. O resultado é uma página mais limpa, moderna e responsiva.

## 2. Página de Ofertas / Encartes

Durante a análise do site original, foi identificado que não existia uma página dedicada aos encartes promocionais. Na época do desenvolvimento deste projeto, o botão de ofertas redirecionava o usuário para um serviço externo.

Como proposta de melhoria, foi criada uma página exclusiva para consulta de encartes, inspirada em soluções utilizadas por grandes redes atacadistas.

A nova interface permite que o usuário selecione o **estado**, a **loja** e o **encarte disponível**, visualizando as ofertas diretamente no navegador sem precisar sair do site.

> **Observação:** Os encartes utilizados são apenas ilustrativos, pois não foi possível obter os materiais oficiais durante o desenvolvimento do projeto.

## 3. Página de Lojas

Na versão original, as unidades eram apresentadas por meio de mapas incorporados diretamente à página, resultando em excesso de informações visuais e pouca organização.

No redesign, essas informações foram reorganizadas utilizando **Cards**, exibindo apenas os dados mais relevantes de cada unidade, como endereço, telefone e horário de funcionamento. O mapa continua disponível através de um botão específico, reduzindo a poluição visual e melhorando a experiência de navegação.

# 🎯 Principais Melhorias Implementadas

Durante o desenvolvimento do redesign foram aplicados diversos conceitos de UI/UX e desenvolvimento front-end, entre eles:

* **Design Responsivo:** utilização do sistema de Grid e dos breakpoints do Bootstrap 5 para garantir uma experiência consistente em smartphones, tablets e desktops.

* **Arquitetura da Informação:** reorganização da estrutura de navegação e distribuição do conteúdo, facilitando a localização das principais informações do site.

* **Hierarquia Visual:** aplicação de tipografia moderna, espaçamentos consistentes, contraste adequado e uso estratégico das cores institucionais para destacar informações importantes.

* **Componentização:** utilização de componentes do Bootstrap, como Cards, Badges, Accordions e Botões padronizados, promovendo maior consistência visual em todas as páginas.

* **Nova funcionalidade de Encartes:** desenvolvimento de uma página exclusiva para consulta de ofertas, permitindo selecionar estado, loja e encarte, aproximando a experiência oferecida por grandes redes do varejo.

* **Código Organizado:** separação entre estrutura (HTML), estilos (CSS), lógica (JavaScript) e dados (encartes.js), facilitando futuras manutenções e expansões do projeto.

* **Boas práticas de Desenvolvimento Web:** utilização de HTML semântico, reaproveitamento de componentes do Bootstrap e redução da quantidade de CSS personalizado para melhorar a manutenção e a escalabilidade do projeto.
